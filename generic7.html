COVID-19 Vaccination Passport

The COVID-19 Vaccination Passport is a web application that was used to track COVID-19 Vaccinated Patients. It was developed in Python and deployed to Flask to connect with a Front-End via a REST API. The Login and Registration infromation was stored in a MongoDB Database and AWS S3 Bucket was used to store the profile pictures of registered patients. Chart.js was used to build interactive charts for data visualization.
