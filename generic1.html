Medical School Dashboard

The Medical School Dashboard was a way for pre-med students to be able to visualize their chances of getting into med school. Data was scaped from online using Python and BeauitfulSoup and stored in a csv file. Then, PySpark DataFrames were used to do some data cleaning/preprocessing and ETL. Then, the cleaned data was stored in an MS Excel spreadsheet. After this, the excel file was imported into Tableau to create an expansive Tableau Dashboard that is user-friendly and interactive.
